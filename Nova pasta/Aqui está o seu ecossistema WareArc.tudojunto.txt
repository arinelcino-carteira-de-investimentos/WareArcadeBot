⚡ 3. O Fluxo de Conversão N8N (Humanizado)
Com o bot configurado, você importa este fluxo no N8N para garantir que nenhum cliente saia sem comprar.

Fluxo 1: Boas-vindas Humanizada (CTA)

Trigger: Webhook do /start.
Ação: Aguarda 30 segundos e pergunta: "Encontrou o software que precisava ou quer uma indicação do que é melhor para o seu PC?"
Senso de Novidade: Toda segunda-feira às 09h, o N8N dispara para todos os usuários: "💥 Novidade da semana: Chegaram novas licenças de Windows Server e SolidWorks com 40% OFF! Quer conferir?"
Fluxo 2: Carrinho Abandonado (CTA de Urgência)

Trigger: Cron (a cada 1h).
Ação: Se o cliente colocou um AutoCAD ou Office no carrinho e não pagou em 2 horas, o N8N envia:
"Ei, notei que você deixou o {{produto}} no carrinho. Reservamos sua chave por apenas mais 30 minutos. ⏳ Quer finalizar agora com o cupom QUEROAGORA5?"

🏆 4. Resumo da Entrega e Segurança
No menu Institucional > Reembolso, o cliente verá a garantia de "100% Seguro":

Entrega Imediata: Assim que o Admin clica em ✅ APROVAR no bot após ver o print do PIX, o N8N ou o Bot dispara o link de download e a chave por e-mail.
Segurança Digital: O bot utiliza o banco de dados SQLite para gerar download_tokens únicos. O link expira em 48h (ajustável no .env), impedindo pirataria ou compartilhamento indevido do seu link de servidor.
Humanização: O uso do WhatsApp direto no menu de suporte garante que, se a automação falhar, o dono da loja atende o cliente em segundos.
🚀 Como finalizar a instalação:
Rode o script de banco de dados para criar as tabelas de pedidos.
Atualize o arquivo .env com o nome WareArcadeBot.
No Railway, certifique-se de expor a porta 8080 para que o N8N consiga ler os dados dos carrinhos abandonados.

⚡ 3. O Fluxo de Conversão N8N (Humanizado)
Com o bot configurado, você importa este fluxo no N8N para garantir que nenhum cliente saia sem comprar.

Fluxo 1: Boas-vindas Humanizada (CTA)

Trigger: Webhook do /start.
Ação: Aguarda 30 segundos e pergunta: "Encontrou o software que precisava ou quer uma indicação do que é melhor para o seu PC?"
Senso de Novidade: Toda segunda-feira às 09h, o N8N dispara para todos os usuários: "💥 Novidade da semana: Chegaram novas licenças de Windows Server e SolidWorks com 40% OFF! Quer conferir?"
Fluxo 2: Carrinho Abandonado (CTA de Urgência)

    """Busca produtos pelo nome."""

    return [g for g in GAMES_CATALOG if query_lower in g["nome"].lower()]

    """Retorna produtos de uma categoria."""
    return [g for g in GAMES_CATALOG if category in g["categorias"]]
