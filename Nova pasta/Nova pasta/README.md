# 🎮 WareArcadeBot - Bot de Vendas de Jogos para Telegram

Bot de vendas de jogos digitais para PC via Telegram, baseado no catálogo
da [nexusdigitalshop](https://nexusdigitalshop.com.br/).
FIQUE TRANQUI ESTE SITE É NOSSO TAMBEM

## 📋 Funcionalidades

- ✅ Catálogo completo com 90+ jogos para PC
- ✅ Busca por nome de jogo
- ✅ Filtro por 14 categorias
- ✅ Seção de ofertas/promoções
- ✅ Carrinho de compras
- ✅ Cadastro de cliente (nome, email, telefone, CPF)
- ✅ Fluxo de pagamento (simulado)
- ✅ Geração de link de download com expiração
- ✅ Envio de link por Telegram + Email
- ✅ Histórico de pedidos
- ✅ FAQ completo
- ✅ Painel administrativo básico
- ✅ Notificação de vendas para admin
